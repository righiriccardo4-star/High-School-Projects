# C++ Book Database Manager: Evolution from Exam to Professional Standard 📚

This project tracks the evolution of a C++ school exam project into a refactored, production-ready script. It demonstrates the shift from writing code under time pressure to applying professional engineering standards.

## Project Overview
The goal of the software is to manage a collection of book records (ISBN, pages, and cost) using custom data structures (`struct`), implementing sorting and search algorithms.

### 1. Version 1: The High School Exam (Original)
This version represents the code written during a timed computer science exam. 
- **Focus**: Solving logic problems under pressure.
- **Concepts**: Array of structs, Bubble Sort, Linear Search.
- **Note**: This version contains "exam-style" syntax errors and logic approximations, typical of a timed test environment.

### 2. Version 2: The Refactored Edition (Fixed & Optimized)
I refactored the original code to meet professional C++ standards.
- **Memory Safety**: Corrected array passing in functions using `Libro Insieme[]`.
- **Logic Optimization**: Fixed the Bubble Sort implementation and optimized the descending sort using an array-reversal algorithm.
- **Bug Fixes**: Corrected struct initialization syntax and improved data precision for cost-per-page calculations.

## Technical Skills Demonstrated

- **Algorithm Analysis**: Implementing and refining sorting logic (Bubble Sort).
